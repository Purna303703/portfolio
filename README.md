# S.Purnachandra Rao - Portfolio

Welcome to my personal portfolio repository! I'm a DevOps fresher who loves working with Linux, AWS, Docker, and CI/CD pipelines. This repo holds the source code for my personal website.

Check out the live version (or just download the code and open it yourself!).

## What's inside?
- **HTML5:** Structuring the content.
- **CSS3:** All the styling, dark theme, and neon glow effects.
- **Vanilla JavaScript:** Powers the interactive star backgrounds, custom cursor, and typewriter text.

## Features
- Cool terminal-style loading screen.
- A moving starfield background on the HTML canvas.
- Fully responsive - works great on mobile and desktop.
- Simple, no frameworks needed.

## Running it locally
Since it's built with pure HTML, CSS, and JS, there's no complicated setup to get it running.

1. Clone the repo:
   ```bash
   git clone https://github.com/Purna303703/portfolio.git
   ```
2. Navigate to the folder:
   ```bash
   cd portfolio
   ```
3. Just double-click on `index.html` to open it in your browser. That's it!
